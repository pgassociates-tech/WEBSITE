## 🚀 Deployment Instructions
1. Push all files (`index.html`, `about.html`, `services.html`, `team.html`, `contact.html`, `privacy.html`, `style.css`) to your GitHub repository.  
2. Go to **Settings → Pages**.  
3. Select branch (`main` or `master`) and root folder (`/`).  
4. Save and access your live site at:  
   `https://<your-username>.github.io/<repository-name>/`
